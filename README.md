# Secure Multi-Room Chat System

Computer Networks Project

## Overview
This project implements a secure multi-client chat system using TCP sockets.  
The system supports chat rooms, private messaging, file transfer, and encrypted communication.

## Features

- Multi-client chat server
- Multiple chat rooms
- Private messaging
- File transfer
- SSL secured communication
- Encrypted secret messaging using Fernet

## Technologies Used

- Python
- TCP Socket Programming
- Threading
- SSL/TLS
- Cryptography (Fernet)
